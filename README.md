# Data2Product
Zoe Hall and Arancha Ramirez  
Data to Product submission  
11 Dec 2025  
  
Directory:  
1. Data Exploration  
   --> Assumption testing  
   --> Data Imputation  
2. Testing prescriptive models  
   --> GDP and OECD mapping data  
   --> Full database creation enabling Forecasting methods  
   --> Initial plots for prescriptive analytics   
3. Exploring Forecasting Methods  
   --> Input datasets to enable approaches (raw or prescriptive)   
   --> Forecast methods comparisons and visualisations   
   --> Core Data to export for the final app  
4. Forecasting system  
   --> Building on forecasting approaches to ensure reliability for prescriptive contexts  
   --> Two test files: TroubleshootingSS.R and TroubleshootingLIV.R  
   --> Includes different forecast model approaches and results  
5. Final App  
   Input datasets:  
     - Prescriptive_data.csv (created in Forecasting system, ARIMA 20 year global forcast)  
     - Final_Country_Waste_Analysis.csv (Full database created in Testing prescriptive models)  
     - Forecasting_data.csv (originally FOR FORECASTING.csv, enables user controlled forecasting by including imputed data)  
     - UN_Trimmed_1988-2024.csv (original UN database, export import and value for plastic waste trade)
